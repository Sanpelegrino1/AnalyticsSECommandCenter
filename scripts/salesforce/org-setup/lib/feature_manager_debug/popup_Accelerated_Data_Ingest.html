Feature Manager

Available Features

Approximate Segment Population

Approximate segment population is calculated using a sample of your profile dataset.

Einstein Segment Creation

Einstein Segment Creation targets specific audiences using natural-language prompts. You first must enable Einstein Generative AI before you can enable this feature. For more information, see Enable Einstein Segment Creation. To track first-time indexing status and optimize results for Einstein Segment Creation use the Metadata Studio.

Navigate the Attribute Library More Intuitively

Opt in to experience improved navigation through the data model hierarchy while building segments. Benefit from improved visual cues that clearly differentiate between available Data Model Objects (DMOs), selectable attributes, and your navigational path.

Notebook AI


Notebook AI is a generative AI notebook where you can upload personal, enterprise, and public content, conduct research, and generate insights.

Segment Preview

Opt in to use Segment Preview in segmentation. Preview the segment before publishing to confirm accuracy, reduce errors, and ensure it targets the right audience.

Semantic Authoring AI

Accelerate and streamline semantic model creation and maintenance, making it faster and easier for data stewards and analysts with capabilities powered by Agentforce.

Accelerated Data IngestBeta

Ingest any data that comes into the Data Cloud DLO layer directly into real-time data graphs without going through any data processing in Data Cloud. Note that when you enable or disable this feature, the change will take effect only at a predetermined time once each day, currently set to 00:00 UTC.
This is a pilot or beta service that is subject to the Beta Services Terms at Agreements-Salesforce.com or a written Unified Pilot Agreement if executed by Customer, and applicable terms in the Product Terms Directory. Use of this pilot or beta service is at the Customer's sole discretion.

Opt in to our improved Data 360 navigation experience. Top tabs are replaced by a hierarchical menu on the left side in which features are grouped by category, providing context and making it easier to find what you need. You can still pin tabs to the top for quick access.
