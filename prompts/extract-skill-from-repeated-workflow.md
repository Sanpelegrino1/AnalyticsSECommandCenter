# Prompt Template: Extract a Repeated Workflow into a Skill

Use this prompt when the same operational task has succeeded twice with nearly the same steps:

"Extract the repeated workflow into a new or updated skill under `skills/`. Use `skills/TEMPLATE.md`. Capture purpose, inputs, prerequisites, exact steps, validation, failure modes, cleanup or rollback notes, and the concrete scripts or tasks to run. Update `skills/INDEX.md` if the skill is added or renamed."
